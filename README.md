# AI-Recommendation-Engine
This project is an AI-powered recommendation engine designed for an e-commerce platform.
The system analyzes user purchase behavior and recommends relevant products based on previous purchases.

The application provides personalized recommendations to improve user experience and increase product discovery.
Features
=> User Login / Signup system
=> Product categories:
   Electronics
   Home
   Fashion
   Beauty
   Sports
=> Add to Cart & Payment
=> Purchase History Tracking
=> Personalized Recommendations
=> “Recommended for You” section based on user purchases
